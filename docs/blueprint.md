# **App Name**: ResolveIt

## Core Features:

- Grievance Submission: Allow citizens to submit new grievances with detailed descriptions and attachments.
- Real-time Status Tracking: Provide real-time updates on the status of filed complaints.
- Resolution Notifications: Notify users when their grievances are resolved or updated.
- Grievance History: Allow users to view a complete history of their submitted grievances.

## Style Guidelines:

- Primary color: Deep blue (#1A4375) for professionalism and trust.
- Background color: Light silver (#D0D3D4) for a clean and modern look.
- Accent color: Green (#4CAF50) for highlighting key actions and positive resolutions.
- Body and headline font: 'PT Sans' for a balance of modern and friendly readability (sans-serif).
- Use clear, professional icons to represent different grievance categories and statuses.
- Maintain a clean and organized layout for easy navigation and readability.
- Use subtle animations to provide feedback on user actions, such as form submissions and status updates.
